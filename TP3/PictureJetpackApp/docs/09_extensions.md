## Aditional features

1. Dark mode
2. Loading progress bar
3. Ability to select up to 5 items as favourites (in a FIFO manner), and a button to view them
4. Cache of up to 50 items (excluding favorites) keeping at least 10 items ahead and 10 behind the current position during navigation (loading Indicator must be relative to what images/items want to be loaded)
5. Offline access to cached items and favorites
